## Installation Guide

This section explains how to install and set up **Robonito** on Windows, macOS, and Linux systems. Follow the instructions for your operating system to get started quickly.


### System Requirements

Before installing Robonito, ensure your system meets the following requirements to guarantee smooth recording and execution of test cases.


#### Operating System

- **Windows:** Windows 10 or later (64-bit)
- **macOS:** macOS 11 (Big Sur) or later
- **Linux:** Ubuntu 20.04 or later (Debian-based distributions recommended)


#### Hardware Requirements

- **Processor:** Dual-core CPU or higher
- **Memory (RAM):** Minimum 8 GB (16 GB recommended for better performance)
- **Disk Space:** At least 5 GB of free space for application files and browser downloads
